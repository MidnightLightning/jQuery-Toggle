jQuery Slide Toggle

Default:
Wide, no animation:
Custom Labels:

Replaces a standard HTML checkbox with an animated slider. Clicking on the slider body or pin toggles the state. The pin can also be dragged across the slider, and upon release will snap to whichever side is closer.

Uses jQuery UI CSS classes for coloration (ui-state-active ui-state-highlight for "on" state, ui-state-default for "off" state), so keeps consistent with other jQuery UI elements on the page.
